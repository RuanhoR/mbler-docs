# Mbler 使用教程

## 安装
确保你已经安装以下工具：
- Node.js 与 npm
- Git
```bash
git clone https://gitee.com/n304sc-haoran/mbler.git
cd mbler
npm install
npm link
```

安装成功后，你可以直接使用：
```bash
# 获取版本
mbler -v
```
