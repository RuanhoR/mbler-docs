# Mbler User Guide

## Installation
Make sure you have installed the following tools:
- Node.js and npm
- Git
```bash
git clone https://gitee.com/n304sc-haoran/mbler.git
cd mbler
npm install
npm link
```

After a successful installation, you can use it directly:
```bash
# Get version
mbler -v
```
