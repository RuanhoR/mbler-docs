# mbler
## Overview
mbler is a dev-tool. It is based on nodejs and typescript. You can use it in windows, android(Install a layer of Termux), linux, mac and more os.  
about 
 - [github repo](https://github.com/RuanhoR/mbler)
 - docs (It's the current page)

## why are we make it
 - It can solve minecraft sapi version chaos
 - It can use the simplest method to make minecraft-addon
 - It can use typescript to make minecraft-addon
 - It can supports i18n
 - You no need to read official JSON tutorial. You only need to write MCX(it very very very hard to read). (This is about [mcx](/guide/mcx) of mbler, It is a beta)
